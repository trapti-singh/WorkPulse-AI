# WorkPulse AI

## Theme
AI at Work: Productivity & Teamwork Reimagined

## Problem Statement
Organizations lose important information across meetings, emails, chats, and documents. Manual follow-ups and task tracking reduce productivity and lead to missed deadlines.

## Solution
WorkPulse AI automatically:
- Summarizes meetings
- Extracts action items
- Assigns task owners
- Sends reminders
- Creates a searchable knowledge base

## Features
- AI Meeting Summaries
- Task Extraction
- Smart Notifications
- Productivity Dashboard
- Knowledge Search

## Tech Stack
- Azure OpenAI
- Microsoft Teams
- FastAPI
- React
- Azure Cosmos DB
- Azure AI Search

## Future Scope
- Multi-Agent Collaboration

- Voice Assistant
